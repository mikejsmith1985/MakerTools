# 🧰 MakerTools

**Free tools that make CNC and 3D printing easier for makers like you.**

No experience needed. No paid subscriptions. Just tools that work.

---

## What's Inside

| Tool | What It Does | Who It's For |
|------|-------------|--------------|
| [🛤 PathMaker](./PathMaker/README.md) | Builds your CNC tool library from Amazon links and generates toolpaths with one click | CNC users with Fusion 360 |
| [🎨 TextureForge](./TextureForge/README.md) | Stamps textures (carbon fiber, knurl, wood grain…) onto any 3D model face | 3D printer users and CNC users with Fusion 360 |
| [💧 MisterWizard](./MisterWizard/README.md) | Walks you through setting up a mist coolant solenoid on your Onefinity CNC | Onefinity CNC users |
| [⚡ WiringWizard](./WiringWizard/README.md) | Plans wiring diagrams and harness layouts for low-voltage projects with ASCII diagrams, connection tables, and BOM recommendations | Makers planning electrical and low-voltage systems |

---

## What You Need First

Before using any of these tools, make sure you have:

- **A computer running Windows** (Mac works for PathMaker and TextureForge too)
- **Python 3.7 or newer** — free download at [python.org](https://www.python.org/downloads/)
  - During install, check the box that says **"Add Python to PATH"** ✅
- **Fusion 360** (free hobbyist license) — for PathMaker and TextureForge only

## How to Install Each Tool

### 🛤 PathMaker (CNC CAM for Fusion 360)

> *"I want Fusion 360 to figure out my CNC toolpaths for me."*

1. Open the `PathMaker` folder
2. Double-click **`install.bat`**
3. Click Next → Next → Finish
4. Open Fusion 360, go to **Tools → Add-Ins**, turn on PathMaker

👉 [Full PathMaker guide →](./PathMaker/README.md)

---

### 🎨 TextureForge (Surface Textures for Fusion 360)

> *"I want my 3D prints or CNC parts to have a cool texture on them."*

1. Open the `TextureForge` folder
2. Double-click **`install.bat`**
3. Click Next → Next → Finish
4. Open Fusion 360, go to **Tools → Add-Ins**, turn on TextureForge

👉 [Full TextureForge guide →](./TextureForge/README.md)

---

### 💧 MisterWizard (Coolant Setup for Onefinity)

> *"I want to set up a mist coolant system on my Onefinity CNC."*

1. Open the `MisterWizard` folder
2. Double-click **`start.bat`**
3. Follow the wizard — it walks you through everything step by step

👉 [Full MisterWizard guide →](./MisterWizard/README.md)

---

### ⚡ WiringWizard (Wiring Diagrams for Low-Voltage Projects)

> *"I want to plan wiring harnesses and electrical layouts for my project."*

1. Open the `WiringWizard` folder
2. Double-click **`start.bat`**
3. Follow the wizard to create your wiring diagram

👉 [Full WiringWizard guide →](./WiringWizard/README.md)

## About MakerTools

MakerTools is a collection of free, open-source tools built for hobbyist makers who use CNC machines and 3D printers. The goal is to remove the hard parts so you can spend more time making.
